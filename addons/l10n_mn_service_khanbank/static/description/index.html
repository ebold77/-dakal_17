Key Features

  • Do synchronization Khanbank account statements:
    • Import bank statements automatically.
    • Update Khanbank Bank Account‘s balance.
  • Make following types of payment from your Khanbank account:
    • Mongolian domestic bank transfer payment from Odoo (Warning: Khanbank recommends that SWIFT transactions should be made via internet banking. So here is no international bank transfer feature).
  • Register Khanbank payment from following parts:
    • Payment form
    • Invoice/Bill register payment form
    • Expense register payment form
  • The features are only visible to “Accounting & Finance / Advisor with Khanbank payment options” user group.

Prerequisites

  • Your Odoo network must have a secure connection to a Khanbank corporate gateway according to Khanbank guide.
  • Install cryptography library in odoo server.

    • You can get it from https://pypi.org/project/cryptography

Setup

1. Go to Accounting>Configuration>Settings menu.

2. Once the setup is complete, you can get the bank parameter information.

Usage

Updating bank accounts

Accounting Dashboard view

Khanbank Payment registration form

Payment can be made directly by Khanbank.

Warning: Once the payment is made by Khanbank, there is no way to cancel this payment. If you have done wrong transaction, you should contact to your Khanbank respondent.
